Detailed Description

SnowboyManKwsNode is a node for manual doa. Design for user who need to get the speaker direction with other method(e.g. with camera) and set that direction to this node to pick the beam in that direction. Please note that, this node can only uplink to VepAecBeamformingNode, and is_single_beam_output should be false.
